# Signature definitions and gene coverage source data

Generated by `source_data_formalization_20260601/create_signature_source_tables.py`. This directory was originally generated under the legacy label `SourceData S15`; the current manuscript points to Supplementary Tables S1--S3 and this traceable source-data directory instead of using the old figure numbering.

This folder formalizes the gene signatures and gene-availability audits requested during CNS-style review.

## Contents

- `Supplementary_Table_S2_signature_definitions_and_gene_coverage.xlsx`: Excel workbook for submission.
- `Supplementary_Table_S3_spatial_and_10x_dataset_accessions.xlsx`: public spatial and 10x dataset accession/source table for submission.
- `Supplementary_Table_S3_spatial_and_10x_dataset_accessions.csv`: CSV version of Supplementary Table S3.
- `signature_definitions.csv`: gene lists and scoring notes for all signatures.
- `multi_sample_spatial_gene_coverage_reconciled.csv`: scored spatial coverage with legacy `TLS_29_imprint` rows relabeled as the actual spatial expression-overlap imprint.
- `multi_sample_spatial_gene_coverage_dataset_summary.csv`: dataset-level coverage summary.
- `multi_sample_spatial_marker_availability_extended.csv`: per-sample key marker availability for ITGAX/CD11C, HLA-family genes, CD74, B2M, CIITA/NLRC5, and mregDC markers.
- `multi_sample_spatial_marker_availability_summary.csv`: dataset-level marker availability summary.
- `xenium_program_gene_availability.csv`: copied Xenium program-gene availability audit.
- `tcga_xenium_signature_availability_legacy.tsv`: copied legacy TCGA/Xenium signature availability table used to audit patient-level 29-gene imprint coverage.
- `signature_label_crosswalk.csv`: explains legacy label harmonization.

## Scope and caveats

The TLS definition in the manuscript remains operational: 12-chemokine score + TLS imprint + spatial co-localization/distance-gradient support. The IF panel identifies TLS-in/near mregDCs and does not define TLS by itself.

The patient-level TLS imprint contains 29 genes. Broad multi-sample spatial scripts used an expression-overlap TLS imprint under the legacy label `TLS_29_imprint`; this is now explicitly relabeled as `TLS_spatial_imprint_overlap` rather than hidden.

Rows in this folder are source-data support and should be interpreted with the Methods and figure legends.

Supplementary Table S3 lists the 23 public multi-sample spatial samples used for Fig. 5/Fig. S4, including GSE/GSM identifiers where applicable, and the public 10x Visium, scFFPE, and Xenium Prime ovarian cancer dataset URLs and software-version notes.
