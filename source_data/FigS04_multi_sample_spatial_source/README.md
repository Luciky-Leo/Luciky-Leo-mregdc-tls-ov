# Source Data for Supplementary Figure S4

Supplementary Fig. S4 contains the multi-sample spatial transcriptomic source-data validation panels after Supplementary Fig. S3 was reserved for the Xenium spatial-map audit.

Source data:
- `multi_sample_spatial_extended_selected_map_cohigh_overlay.csv`: plotted spot coordinates and classes for representative spatial maps.
- `multi_sample_spatial_extended_selected_map_samples.csv`: selected representative samples and sample-level metrics.
- `multi_sample_spatial_extended_sample_metrics_long.csv`: sample-level association and co-high enrichment metrics.
- `multi_sample_spatial_extended_gene_coverage_fraction.csv`: per-sample gene-set coverage audit for spatial modules.
- `multi_sample_spatial_extended_exact_marker_availability.csv`: HLA-DRA, LAMP3, and CCR7 availability by sample.

The removed panels from the previous layout were the dataset-size overview, dataset-level support summary, and TLS-decile gradient plot. The retained/rebuilt figure focuses on source-data support panels for multi-sample spatial validation.
