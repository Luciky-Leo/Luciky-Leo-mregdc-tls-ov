# Source Data for Supplementary Figure S3

Supplementary Fig. S3 contains user-revised Xenium spatial-map panels used as visual support for the spatial feature audits in Supplementary Figs. S1 and S2.

Files:
- `FigureS3_Xenium_spatial_maps-re.pdf`: active user-revised figure included in the Supplementary Information.
- `singleCellHaystack_spatial_maps_user_screenshot.png`: source screenshot for singleCellHaystack-related spatial maps.
- `Hotspot_spatial_maps_user_screenshot.png`: source screenshot for Hotspot-related spatial maps.
- `README_manifest.txt`: generation manifest from the earlier assembled S3 draft.

Interpretation:
These maps are descriptive spatial QC/audit visualizations for the Xenium subset. They are not used to define histological TLS maturity.
