# v1.1.0 - Cancers submission version

This release updates the reviewer-facing reproducibility archive for the
Cancers submission version of the ovarian cancer TLS/mregDC manuscript.

## Main updates

- Replaced the manuscript source with the Cancers/MDPI LaTeX submission version.
- Updated the compiled manuscript PDF and submission precheck report.
- Replaced Fig. 1 with the latest author-revised workflow figure.
- Synced Fig. 1-Fig. 11 review/upload figure PDFs.
- Synced Supplementary Figures S1-S6 and Supplementary Tables S1-S3.
- Synced the current Source Data archive with the revised Fig. 2 source-data
  note and target-journal source-data wording.
- Updated repository README and CITATION metadata from the previous JTM-facing
  version to the Cancers submission version.

## Scope notes

- The IF image layer remains representative merged-image support and is not used
  to claim tissue-level cell density or single-cell four-marker co-localization.
- Public spatial transcriptomic antigen-presentation readouts remain
  MHC-II/AP proxy signals where exact HLA genes are absent.
- Prognostic modeling, virtual perturbation, and drug prioritization remain
  hypothesis-generating and require independent validation.
- Cover letter and private submission-only portal files are not included in this
  public release.
